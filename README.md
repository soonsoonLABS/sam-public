# SAM Public

Public tools, installers, and documentation for connecting supported clients to
SoonSoon AI Management (SAM).

## Start Here

Start with [`00-sam-setup/`](00-sam-setup/) to enter your SAM API key, confirm
only the key prefix, and run a tiny `Hello SAM` test call.

## 처음 시작

먼저 [`00-sam-setup/`](00-sam-setup/)에서 SAM API 키를 입력하고, 키
앞부분만 확인한 뒤, `Hello SAM` 테스트 호출까지 진행하세요.

## Available Areas

- [`00-sam-setup/`](00-sam-setup/): first SAM environment setup for macOS and
  Windows, with Korean and English commands.
- [`code-agent/`](code-agent/): coding-agent connection tools and guides for
  Codex and Claude Code.

## Security

This repository contains no SAM server code, production configuration, or
credentials. Never add API keys to files, issue text, command history, or URLs.
